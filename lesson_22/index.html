Домашнє завдання JS

Завдання №1 (Структура коду. Інструкції. Режим use strict.)

Помилку визве варіант №3 - немає значка ";" після інструкцій. При написанні інструкцій в рядок, це викликає помилку.

Помилку визве варіант №4 - в інструкції не закриті одинарні лапки, що викликає помилку.

Завдання №2 (Змінні в JAVASCRIPT. Константи. Ключові слова LET CONST VAR.)

Варіант №1 записаний не вірно - немає ";" між оголошенням змінних, що визве помилку в консолі.

Варіант №3 записаний не вірно - таким чином оголошується константа, яка вже відома до початку вонання скрипту, тому логічно вона не може визначатися арифметичною дією.

Завдання №3 (Null String Undefined Boolean Number)

Варіант №1 записаний не вірно - при такому записі в консоль виведеться ім'я змінної userAge, для виводу значення змінної userAge потрібно поставити зворотні лапки "``".

Варіант №2 записаний не вірно - значення буде Infinity, так як діленняна 0 дає нескінченність.

Варіант №3 записаний не вірно - тип даних буде undefined, так як змінній не присвоєне ніяке значення.
